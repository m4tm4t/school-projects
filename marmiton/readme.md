# Marmiton - robard_m

## Project dependencies

  * php 5.4+
  * postgresql 9.3+
  * nodejs, npm, bower & gulp

## Install dependencies

```shell
$ npm install
$ bower install
```

## Create database

```shell
$ createdb marmiton
$ psql marmiton < schema.sql
```

## Launch

```
$ gulp
```

Then visit http://127.0.0.1:4242
